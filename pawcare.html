PawCare Project

Description: PawCare is a full-stack web application that allows pet owners to find, book, and review services like grooming, vet care, and sitting. The project includes customer and provider dashboards, secure login, and review functionality.

Technologies: Java, Spring Boot, HTML, Freemaker, CSS, XAMPP

Team Members: Yasmine Chiboub (customer frontend/backend), Libby (provider/backend logic), Sajid (Admin/backend)
